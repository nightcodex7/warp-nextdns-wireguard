Installation Guide

Get WARP + NextDNS Manager up and running on your system

Quick Start

WARP + NextDNS Manager is a cross-platform tool that combines Cloudflare WARP's encrypted tunneling with NextDNS's advanced DNS filtering. Follow the installation guide for your platform below.

Prerequisites

  • Python 3.8 or higher
  • Administrator/root access
  • Internet connection
  • NextDNS configuration ID (optional)

Installation Methods

Python Package Installation

1

Clone the Repository

Terminal 
git clone https://github.com/nightcodex7/warp-nextdns-wireguard.git
cd warp-nextdns-wireguard
2

Install Dependencies

Terminal 
pip install -r requirements.txt
3

Run the Setup

Terminal 
python cli.py setup

Git Clone Method

1

Clone and Setup

Terminal 
git clone https://github.com/nightcodex7/warp-nextdns-wireguard.git
cd warp-nextdns-wireguard
pip install -r requirements.txt
python cli.py setup

Pip Installation

1

Install via Pip

Terminal 
pip install warp-nextdns-manager
2

Run Setup

Terminal 
warp-nextdns setup

Configuration

After installation, you can configure WARP + NextDNS Manager with your NextDNS configuration ID for enhanced filtering capabilities.

Configuration 
python cli.py config --nextdns-id YOUR_CONFIG_ID

Usage

Once installed and configured, you can use the following commands:

Basic Commands 
python cli.py status    # Check service status
python cli.py start     # Start WARP + NextDNS
python cli.py stop      # Stop services
python cli.py test      # Run connectivity tests
python cli.py logs      # View logs
