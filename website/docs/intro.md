# Introduction

Welcome to **WARP NextDNS Manager** - a powerful tool that combines Cloudflare WARP's secure tunneling with NextDNS's advanced DNS filtering capabilities.

## What is WARP NextDNS Manager?

WARP NextDNS Manager is an automated setup and management tool that enables you to:

- 🔐 **Encrypt your internet traffic** through Cloudflare WARP using WireGuard protocol
- 🛡️ **Filter DNS requests** using your custom NextDNS profile
- 🚀 **One-click setup** with automatic privilege elevation
- 🖥️ **Cross-platform support** for Linux, Windows, and macOS
- ⚙️ **Service management** with boot persistence

## Why Use This?

### Benefits of WARP
- Free VPN-like tunnel over WireGuard
- Encrypts traffic and masks IP
- Low latency, ideal for gaming
- Bypasses basic network restrictions

### Benefits of NextDNS
- DNS-level ad and tracker blocking
- Malware and phishing protection
- Parental controls and analytics
- Custom filtering rules

### Combined Power
By routing traffic through Cloudflare's WARP tunnel while resolving DNS via your NextDNS profile, you get the best of both worlds - enhanced privacy, security, and control over your internet connection.
